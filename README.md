# 🏦 ArixBank - ASP.NET Banking System

A web-based banking application built using ASP.NET WebForms, C#, Bootstrap, and SQL Server.

## 🌐 Live Demo
You can view and test the live application here:
👉 **[http://arixbank.somee.com/SignIn_Up/Welcome.aspx](http://arixbank.somee.com/SignIn_Up/Welcome.aspx)**

---

## 🚀 Features
- **User Authentication:** Safe Register and Login system.
- **Account Dashboard:** View account details, current balance, and quick actions.
- **Deposit & Withdraw:** Secure transaction processing.
- **Fund Transfer:** Send money securely to other registered accounts.
- **Transaction History / Statements:** Track all deposits, withdrawals, and sent/received money.
- **Settings:** Change account passwords and update user details.

---

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** C# (ASP.NET WebForms)
- **Database:** MS SQL Server (hosted on Somee)
- **Server Environment:** Windows Server IIS

---

## 📂 Folder Structure
- `Pages/` - Contains dashboard, transactions, statements, and settings pages.
- `SignIn_Up/` - Contains the Welcome, Login, and Register pages.
- `Content/` - CSS files including Bootstrap styles.
- `Scripts/` - JavaScript files.
- `App_Data/` - Database files (Local development backup).
